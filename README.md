# 💖 Puzzle do Amor

Este é um projeto muito especial, criado para comemorar **5 meses de namoro**.  
Um site romântico e interativo, feito com muito carinho, para celebrar cada momento que passamos juntos.

No site você vai encontrar:

- 🎶 Música de fundo: *"Minha Namorada – Ferrugem"*  
- 📸 Foto do casal, para sempre lembrar desses dias especiais  
- 💌 Uma carta gigante e emocionante, cheia de sentimentos e promessas  
- ✨ Nossos momentos mais importantes, lembranças que marcaram nossos 5 meses  
- 🏴‍☠️ Referências a One Piece, simbolizando nossa aventura juntos  
- ❤️ Corações animados caindo na tela, trazendo leveza e amor  
- 🎵 Animações suaves e efeitos para deixar tudo mais bonito
